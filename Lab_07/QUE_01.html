Question 4: Six Heading Levels

H2 - Major section headings

H4 - Sub-subsection headings

H5 - Minor headings
H6 - Smallest headings for fine details

Question 11: Nested Lists for Recipe

Chocolate Chip Cookies Recipe

Ingredients:

  • Dry Ingredients:
    • 2 cups all-purpose flour
    • 1 tsp baking soda
    • 1 tsp salt
  • Wet Ingredients:
    • 1 cup butter, softened
    • 3/4 cup brown sugar
    • 1/2 cup white sugar
    • 2 large eggs
  • Add-ins:
    • 2 cups chocolate chips
    • 1 cup chopped nuts (optional)

Instructions:

  1. Preheat oven to 375°F
  2. Mix dry ingredients:
    1. Combine flour, baking soda, and salt
    2. Set aside
  3. Prepare wet mixture:
    1. Cream butter and sugars
    2. Add eggs one at a time
    3. Mix well
  4. Combine and bake:
    1. Gradually add dry ingredients
    2. Fold in chocolate chips
    3. Drop on baking sheet
    4. Bake 9-11 minutes

Question 12: Definition Lists for Glossary

HTML/CSS Glossary

HTML
HyperText Markup Language - the standard markup language for creating web pages
CSS
Cascading Style Sheets - used for describing the presentation of a document written in HTML
JavaScript
A programming language that enables interactive web pages and is an essential part of web applications
DOM
Document Object Model - a programming interface for HTML and XML documents
Responsive Design
An approach to web design that makes web pages render well on a variety of devices and window sizes
Semantic HTML
HTML that introduces meaning to the web page rather than just presentation
